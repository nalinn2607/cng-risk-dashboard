# CNG Risk Dashboard
A real-time AI/ML-powered risk strategy dashboard for a non-banking subprime lender, dealing in payday, installment, title auto, and pawn loans.

## Features
- **Portfolio Overview:** Monitor active loans, AUM, and key metrics.
- **Early Warning Signals:** High-risk event detection (NSF events, call avoidance, loan stacking).
- **DPD Migration:** Roll rates and cohort tracking for delinquency buckets.
- **Charge-Off Prediction:** Customer-level risk scoring and explanations (SHAP).
- **Estimated Loss Forecast:** CECL-compliant loss curves and expectations.
- **Behavioural Segmentation:** Unsupervised clustering of risk vs engagement.
- **Dynamic Themes:** Frosted glass light theme and premium dark mode.

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Chart.js (for visualizing risk and migrations)

## Deployment (Vercel)
This project is fully static and can be deployed directly to Vercel without a build step.
